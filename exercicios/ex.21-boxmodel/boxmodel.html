Modelos de caixa

Modelos de caixa são formas do html e css de organizar o conteúdo através do aninhamento. Para configurar as caixas usamos parâmetros como, padding, margin, border

Box-level

Box level é um dos modelos de caixa, cujo qual usa tags que a caixa pula para a linha de baixo e ocupa toda a tela usando tags como ( div, p, h1).
